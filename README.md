# Stopwatch
A stopwatch created with C++

## Features
* Toggle
* Stop
* Reset
* Get the split time

## How to Run
In a terminal/command prompt, to use the makefile, type:
```
make
```

To run the executable:
```
./stopwatch
```

To clean up the object files and executable:
```
make clean
```
